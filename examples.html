Примеры

1. Чтобы получить и вывести расписание на эту неделю для определенного клуба используйте следующи код:

    //Подключаем библиотеки
    $assets = MODX_BASE_PATH . 'assets/modules/schedule/';
    require_once $assets . 'includes.php';
    //Получаем расписание на эту неделю
    $thisWeekSchedule = DataGetter::schedule2('01', 'this week');
    //Получаем список правил(ограничений) для вывода расписаний
    $restrictions = TrainingRestriction::makeArrayFromActive();
    //Применяем правила
    $thisWeekSchedule->applyRestrictions($restrictions);
    //Выводим расписание
    echo $thisWeekSchedule->toHTML();

Выведится большая и нестилизованная таблица с расписанием

Где '01' это id клуба на качалова, 'this week' дает понять что расписание нужно на эту неделю. Если нужно расписание на следующую нужно написать 'next week'.
